# 🍽️ Swiggy Sales Analysis Dashboard | Microsoft Excel

An interactive **Swiggy Sales Analysis Dashboard** built entirely in **Microsoft Excel** using Pivot Tables, Pivot Charts, Slicers, Conditional Formatting, and Excel Dashboard techniques. The dashboard helps analyze sales performance, customer ratings, order trends, regional performance, and food category insights.

---

# 📌 Project Overview

This project transforms raw Swiggy sales data into an interactive dashboard that enables users to monitor key business metrics and identify sales trends across different states, cities, food categories, restaurants, and time periods.

The dashboard provides meaningful business insights that support data-driven decision-making through easy-to-understand visualizations.

---

# 🎯 Project Objectives

- Analyze overall sales performance.
- Monitor monthly, weekly, and daily sales trends.
- Compare sales across different states and cities.
- Analyze food category performance.
- Evaluate customer ratings and total orders.
- Identify top-performing locations.
- Build an interactive dashboard using Microsoft Excel.

# 💡 Key Insights

- Generated **₹53.01M** in total sales with over **197K orders**.
- The average customer rating remained **4.34**, indicating strong customer satisfaction.
- Monthly sales showed consistent performance with only minor fluctuations.
- Non-Veg items contributed a larger share of total sales compared to Veg items.
- Bengaluru emerged as the highest revenue-generating city among the top five cities.
- Certain states consistently outperformed others in total sales, highlighting strong regional demand.
- Interactive filters allow users to analyze sales by month, category, and restaurant for deeper business insights.
